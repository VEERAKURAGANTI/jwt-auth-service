JWT Flow
how it works
1
Client sends credentials to /api/auth/login
2
Server returns accessToken (15m) + refreshToken (7d)
3
Client sends Authorization: Bearer <token> on every request
4
On 401 → call /api/auth/refresh for new token
5
On logout → refresh token deleted from DB
